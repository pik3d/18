Фреимворк  pik3d  v.18.1

     ОБРАБОТКА и 3D-ОТОБРАЖЕНИЕ  МАТРИЦ/ КАДРОВ и их ПОСЛЕДОВАТЕЛЬНОСТЕЙ/ СТЭКОВ

При исследовании свойств изменяющейся во времени поверхности, удобно хранить,
обрабатывать и отображать совокупность изображений(кадров/матриц), как одно целое.
Такую последовательность называют СТЭКОМ изображений( возможно из одного кадра).

Фреимворк предлагает стандарт хранения/обмена данными: ###-формат и базовый набор
команд обработки и отображения стэков. Для преобразования текста матрицы в ###-формат
требуется добавить строку заголовка:

	### имя_матрицы [ m ,n ] ,    где:  m-кол.строк,  n-столбцов

Для чтения ###-формата в редакторе ImageJ реализован плагин: ###_Reader_ImageJ.jar.
Для чтения и записи матриц в PYTHON - package: kadr.py.

Фреимворк имеет простые но эффективные средства 2D-графики для отображения
строк и столбцов матриц. Изображения сохраняются в векторном формате:
WMF - Window Meta File. Они хорошо масштабируются и их можно использовать
в MS Word, просматривать при помощи ACDSee.

Элемент матрицы можно отобразить как точку в 3D-пространстве:
X -номер строки матрицы, Y -номер столбца, Z -значение элемента. Совокупность
таких точек образует 3D-поверхность. Визуализация поверхности с простым управлением
ракурсом, подсветкой, цветом и т.д. составляет функционал команды: s3d -"Surface-3D".
Просматривать стэк можно в режиме "кадр за кадром", как видео-ролик.

Команды фреимворка - это исполняемые(.exe) модули и их набор не фиксирован.
Использована технология лончеров("launch"- запускать, начинать),
которые инкапсулируют сложную стартовую конфигурацию JAVA-машины.
Фреимворк имеет простые средства создания новой команды на JAVA. Дополнительные
команды могут быть реализованы на любом удобном алгоритмическом языке.

В простом варианте программа обработки представляет собой скрипт( .bat или .cmd).
От команды к команде данные передаются в виде файлов ###-формата или средствами
"pipe-line" - конвейера( это экономит время и дисковое пространство ).

Фреимворк можно установить на  Windows 7, 10.
Версия 18.1 скомпилирована JAVA-17 с использованием javaFX.
Установка JAVA не требуется, достаточно развернуть её в какой-нибудь папке.

Описание фреимворка, форматов и процедуры установки в файле:
pik3d_DOC.html (содержится в установочном архиве).
